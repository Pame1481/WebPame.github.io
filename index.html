SUMMARY: LATITUDE vs. X

"For this weather project, we created a python script to show the weather conditions in more than 500 cities around the globe of varying distance from the equator (different latitudes).
In order to develop this project, we used a simple Python Library and OpenWeather API to generate the dataframe of the numerous cities and we further used Matplolib to derive the relationship between Latitude and 4 factors (Max Temperature, Humidity, Cloudiness, Wind Speed).
This site provides four scatter plots of the variables mentioned before, a comparison of the variables and the data generated undertaking this project.
